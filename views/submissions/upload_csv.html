Submission File Upload

  • This page is for uploading a file (csv format) of sample data for submission to the Marine Biological Laboratory's KECK sequencing facilty.
  • To start the submission process and download a template file you should go to the submission request page.
  • We recommend you use OpenOffice or a Microsoft Excel spreadsheet to edit the downloaded file.
  • Make sure to keep the same headers, in the correct order and save it as a UNIX csv file.
